# Personal Restaurant Guide

**COMP3074 - Mobile App Development I**

## Project Overview
The **Personal Restaurant Guide** is a mobile application designed to help users track the restaurants they visit, rate them, and share information. It leverages Android tools like SQLite for local storage and the Google Maps API for GPS integration.

## Features
- Add, edit, and remove restaurant information:
  - Name, Address, Phone, Description, Tags, and Rating.
- Search for restaurants by name or tags.
- View restaurant location on a map and get navigation directions.
- Share restaurant details via email and (optionally) Facebook/Twitter.
- Responsive and user-friendly UI design.

## Technologies
- **Android Studio**: Java/Kotlin for app development.
- **SQLite**: Local storage for restaurant data.
- **Google Maps API**: For location and navigation functionality.
